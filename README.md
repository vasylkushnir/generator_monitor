# Generator Monitor / Моніторинг Генератора ⚡🔋

[![React Native](https://img.shields.io/badge/React_Native-18.x-61DAFB?logo=react&logoColor=white)](https://reactnative.dev/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?logo=typescript&logoColor=white)](https://www.typescriptlang.org/)
[![i18next](https://img.shields.io/badge/i18next-Localized-009688?logo=i18next&logoColor=white)](https://www.i18next.com/)

An Android application designed to monitor generator runtime and track oil service intervals via ESP32 API integration.
Додаток для Android, призначений для моніторингу часу роботи генератора та відстеження інтервалів заміни оливи через інтеграцію з API на базі ESP32.

---

## ✨ Features / Можливості

- **Real-time Monitoring**: Track total work hours and current status. (Моніторинг у реальному часі: загальний час роботи та поточний стан).
- **Oil Resource Tracking**: Visual progress bar showing remaining oil life. (Відстеження ресурсу оливи: візуальний прогрес-бар залишку ресурсу).
- **Service History**: Log and view previous maintenance records. (Історія обслуговування: логування та перегляд попередніх замін оливи).
- **Fully Localized**: Support for English and Ukrainian languages based on system settings. (Повна локалізація: підтримка англійської та української мов).
- **Modern UI**: Clean dark-themed design with adaptive icons. (Сучасний UI: мінімалістичний темний дизайн, адаптивні іконки).

---

## 🛠 Tech Stack / Технологічний стек

- **Frontend**: React Native (TypeScript)
- **State & Data**: React Hooks, SWR / Fetch API
- **Localization**: i18next + react-i18next
- **Hardware Integration**: ESP32 (HTTP Server / JSON API)
- **Storage**: AsyncStorage / MMKV
